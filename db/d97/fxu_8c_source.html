42  Synopsis [Performs fast_extract on a set of covers.]
43 
44  Description [All the covers are given in the array p->vSops.
45  The resulting covers are returned in the array p->vSopsNew.
46  The entries in these arrays correspond to objects in the network.
47  The entries corresponding to the PI and objects with trivial covers are NULL.
48  The number of extracted covers (not exceeding p->nNodesExt) is returned.
49  Two other things are important for the correct operation of this procedure:
50  (1) The input covers do not have duplicated fanins and are SCC-free.
51  (2) The fanins array contains the numbers of the fanin objects.]

157  }
158 
159  if ( pData->fVerbose )

161  p->nDivs1, p->nDivs2, p->nDivs3 );
162 
163  // create the new covers
